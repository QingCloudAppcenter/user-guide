# 青云QingCloud AppCenter 应用使用指南

## 简介

此文档收集了青云QingCloud 在 AppCenter 提供的云应用的使用文档，绝大部分是基于开源项目且经过配置优化的应用，具有一键部署、弹性伸缩和丰富的监控告警等云计算特性。帮助用户更好地管理基础资源，让用户从繁琐复杂的日常基础组件的运维中解放出来，把时间和精力更多的投入到自己核心的业务创新上面。
